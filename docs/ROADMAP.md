# Дорожная карта

Честный статус: что уже работает, что в виде каркаса, что впереди.

## ✅ Реализовано (0.1)

- Математика: `Vector2/3/4`, `Vector2I`, `Color`, `Quaternion`, `Basis`,
  `Transform2D/3D`, `Rect2`, `Aabb`, `Mathf`.
- Система объектов: `GObject`, сигналы, `[Export]`, `ClassDb` (рефлексия).
- Дерево сцены: `Node`, `SceneTree`, жизненный цикл, группы, deferred-free.
- 150+ типов узлов (2D, 3D, UI, физика, аудио, анимация, навигация, VFX,
  террейн, Grease Pencil).
- Сериализация: `.scen`, `.material`, `.shadergraph`, `project.gimnasy`.
- Нодовый шейдер-граф → компиляция в GLSL (C# и эталон на Python).
- Террейн с кистями (Add/Subtract/Grow/Erode/Smooth/Flatten/Paint/SeaLevel).
- Скульптинг мешей (7 кистей) и операции редактирования (subdivide/extrude/smooth).
- Grease Pencil (штрихи, слои, упрощение RDP).
- VFX-узлы: дым, огонь, вода, облака, туман, взрывы, трейлы, молнии, погода,
  пост-эффекты.
- Игровой цикл с фиксированным шагом физики; headless-рендер.
- C#-скриптинг через подклассы узлов + загрузка сборки игры.
- Python-инструменты: валидация, импорт, иконки, упаковка (Win/Linux/macOS).
- 19 юнит-тестов инструментария (проходят).

## 🚧 Каркас / упрощённо

- **Рендеринг** — только headless (`NullRenderingServer`). Команды отрисовки
  считаются, но не выводятся на экран.
- **Физика 3D** — `CharacterBody3D.MoveAndSlide` использует плоский пол;
  широкофазного солвера ещё нет (в 2D — посекторный sweep по AABB).
- **VFX/террейн/Grease Pencil** — полноценные данные и алгоритмы, но финальная
  визуализация ждёт GPU-бэкенд.
- **Xcode-проект** — генерируется как External Build System (вызывает
  `dotnet publish`), не нативная сборка.
